# Linux-Ubuntu 基本操作



**设备文件名**

| 硬件              | 文件名               |
| ----------------- | -------------------- |
| IDE硬盘           | /dev/hd[a-d]         |
| SCSI/SATA/USB硬盘 | /dev/sd[a-p]         |
| 光驱              | /dev/cdrom或/dev/sr0 |
| 软盘              | /devfd[0-1]          |
| 打印机(25针)      | /dev/lp[0-2]         |
| 打印机(USB)       | /dev/usb/lp[0-15]    |
| 鼠标              | /dev/mouse           |
