# 自然语言处理

## 情感分类

### 情感分类的类别

1. 文档级别(新闻)
2. 句子(片面级别, 观点)
3. 细粒度

## QA 系统

### 流程

#### 基于问答知识库

1. 切词
2. 维护问答知识库(根据问题 feq 排序)
3. 根据相似度排序
4. 返回给用户

#### 基于知识图谱

1. 切词
2. 找关键词(也许是实体发现)
3. 没有完全匹配就需要实体对齐
4. 匹配关系(同样, 可能需要关系对齐)

## Word2Vec

2. 词序无关的: 苹果吃我, 我吃苹果 -> TextCNN
