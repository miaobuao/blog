**设备文件名**

| 硬件                          | 文件名                         |
| -----------------        | --------------------      |
| IDE硬盘                     | /dev/hd[a-d]              |
| SCSI/SATA/USB硬盘 | /dev/sd[a-p]               |
| 光驱                          | /dev/cdrom或/dev/sr0 |
| 软盘                          | /devfd[0-1]                 |
| 打印机(25针)             | /dev/lp[0-2]                |
| 打印机(USB)              | /dev/usb/lp[0-15]       |
| 鼠标                          | /dev/mouse                |



**基本指令**

| 指令     | 作用                                                   |
| -------- | ------------------------------------------------------ |
| date     | 打印当前日期                                           |
| ls       | 显示目录                                               |
| free     | 显示内存状态                                           |
| df [-lh] | 显示磁盘分区                                           |
| du       | （英文全拼：disk usage）命令用于显示目录或文件的大小。 |
| pwd      | 显示当前工作路径                                       |
| cp       | 复制                                                   |
